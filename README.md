# HR Attrition Analysis Dashboard in Power BI

This project is an interactive HR Attrition Analysis Dashboard built in Power BI to analyze employee attrition patterns and workforce trends.

## Key Insights Covered
- Attrition by Job Role
- Attrition by Department
- Attrition by Education Field
- Attrition by Age Group
- Attrition by Salary Slab
- Attrition by OverTime
- Attrition by Tenure

## KPI Metrics
- Total Employees
- Attrition Count
- Attrition Rate
- Average Age
- Average Salary

## Tools Used
- Power BI
- Power Query
- DAX

## Features
- Interactive slicers for Department, Gender, OverTime, and Education Field
- Clean and compact dashboard layout for HR decision-making
- Data transformation and calculated columns for grouped analysis

## Project Outcome
This dashboard helps identify key attrition patterns and supports data-driven HR analysis.
